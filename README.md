# webservice
CSYE6225 Workspace

Assignment#1

Pre-requisites:
- Install Node.js 

Getting Started:
- Clone the repository
    git clone https://github.com/Amal-Mat/webservice.git

- Install Dependencies
    cd webservice
    npm install express
    npm install cors
    npm install dotenv
    npm install -g nodemon
    npm install mocha
    npm install supertest

- Build and run the project
    npm start
